    Features:
-------------------------------------------------------

- Widgets
Wlow has one sidebar on the right.

- Page Templates
Wlow has a home page template with Sections and Parllax

We Have:
Blog Layout, Category, Tag & Search Layout,

- Menus
One menu on the top.
